# MadaMesh - Réseau Maillé Local d'Urgence

## Vision
Connecter Madagascar sans internet. Un réseau P2P décentralisé utilisant Wi-Fi Direct et Bluetooth, propulsé par un moteur C++ ultra-performant.

## Architecture
- **Core (C++)** : Gestion du routage maillé, chiffrement (libsodium), compression et sockets bas niveau.
- **JNI Bridge** : Interface entre le moteur natif et l'UI Android.
- **UI (Kotlin)** : Interface utilisateur moderne et intuitive.

## Fonctionnalités
- [ ] Protocole Mesh P2P (Routage dynamique)
- [ ] Transfert de fichiers ultra-rapide
- [ ] Chiffrement de bout en bout
- [ ] Dashboard d'audit Wi-Fi local
